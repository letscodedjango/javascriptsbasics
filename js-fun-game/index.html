What actually Javascript is? Does it have anything with Java?

Javascript has nothing to do with Java. So dont get confuse. Both are complete different from each other and used for different purposes.

  • Javascript is a lightweight, cross-platform, object-oriented programming language

  • Javascript is one of the three (HTML/CSS) core technologies of any web development.

  • Earlier, Javascript was used as a client side language but thanks to node.js, we can use javascript on server side too.

Javascript make the web appliaction dynamic and interactive. We can interact with our browser/web appliaction using Javascript
